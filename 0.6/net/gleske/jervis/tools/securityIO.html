Package: net.gleske.jervis.tools

[Groovy] Class securityIO

    • net.gleske.jervis.tools.securityIO

    • A class to provide cryptographic features to Jervis such as RSA encryption and base64 encoding.

      Sample usage

      import net.gleske.jervis.tools.securityIO
def security = new securityIO('/tmp',2048)
security.generate_rsa_pair()
println 'Private key path: ' + security.id_rsa_priv
println 'Public key path: ' + security.id_rsa_pub
println 'Key size: ' + security.id_rsa_keysize.toString()
def s = security.rsaEncrypt('hello friend')
println 'Length of encrypted output: ' + s.length()
println 'Encrypted string:'
println s
println 'Decrypted string:'
println security.rsaDecrypt(s)

      • Field Detail

        • public static int default_key_size

          The default key size for id_rsa_keysize. Default: 2048

        • public int id_rsa_keysize

          The key size in which RSA keys will be generated. It is highly recommended this be 1024 bits or higher in powers of two. Default: value of default_key_size.

        • public String id_rsa_priv

          Path to the RSA private key. This will be used by encryptiong and decryption. During key generation it is the location where the private key will be written. Default: /tmp/id_rsa.pem

        • public String id_rsa_pub

          Path to the RSA public key. This will be used by encryptiong and decryption. During key generation it is the location where the public key will be written. Default: /tmp/id_rsa.pub.pem

        • securityIO(String path)

          Instantiates the default value for id_rsa_keysize but sets id_rsa_priv and id_rsa_pub based on path. path sets the directory where the public and private key will be generated. Specifically values are set in the following manner: 

          
id_rsa_priv = checkPath()(path) + '/id_rsa.pem'
id_rsa_pub = checkPath(path) + '/id_rsa.pub.pem'
id_rsa_keysize = default_key_size

          Instantiates setting custom values for id_rsa_priv, id_rsa_pub, and id_rsa_keysize. path sets the directory where the public and private key will be generated. keysize sets the value for id_rsa_keysize. Specifically values are set in the following manner: 

          
id_rsa_priv = checkPath()(path) + '/id_rsa.pem'
id_rsa_pub = checkPath(path) + '/id_rsa.pub.pem'
id_rsa_keysize = keysize

        • void generate_rsa_pair(String priv_key_file_path, String pub_key_file_path, int keysize)

          Generate an RSA key pair (private key and public key). It does not take values from id_rsa_priv, id_rsa_pub, nor id_rsa_keysize. For third party reference, this is essentially executing the following commands in a terminal. 

          openssl genrsa -out /tmp/id_rsa 2048
openssl rsa -in /tmp/id_rsa -pubout -outform pem -out /tmp/id_rsa.pub
          Parameters:
          priv_key_file_path - A file path where the private key will be written on the filesystem.
          pub_key_file_path - A file path where the public key will be written on the filesystem.
          keysize - The key size in bits of the key pair.

        • Boolean isSecureField(def field)

        • String rsaDecrypt(String ciphertext)

          Uses RSA asymetric encryption to decrypt a cipher text String and outputs plain text. For third party reference, this is essentially executing the following commands in a terminal. 

          echo 'ciphertext' | base64 -d | openssl rsautl -decrypt -inkey /tmp/id_rsa
          Parameters:
          ciphertext - A Base64 encoded cipher text String to be decrypted.
          Returns:
          A plain text String or more generically: plaintext = RSAPrivateKeyDecrypt(base64decode(ciphertext))

        • String rsaEncrypt(String plaintext)

          Uses RSA asymetric encryption to encrypt a plain text String and outputs cipher text. For third party reference, this is essentially executing the following commands in a terminal. 

          echo -n 'plaintext' | openssl rsautl -encrypt -inkey /tmp/id_rsa.pub -pubin | base64 -w0
          Parameters:
          plaintext - A plain text String to be encrypted.
          Returns:
          A Base64 encoded cipher text or more generically: ciphertext = base64encode(RSAPublicKeyEncrypt(plaintext))
