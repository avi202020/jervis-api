Package: net.gleske.jervis.tools

[Groovy] Class scmGit

    • net.gleske.jervis.tools.scmGit

    • A class to provide useful functions for interacting with a local git repository.

      Sample usage

      import net.gleske.jervis.tools.scmGit
def x = new scmGit()
print "Git root dir: "
println x.getRoot()
