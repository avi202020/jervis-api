Package: net.gleske.jervis.tools

[Groovy] Class fileIO

    • net.gleske.jervis.tools.fileIO

    • A class to provide basic file operations from readFile() strings in a yaml file. readFile() strings typically have the format of readFile(/path/to/file.sh) where /path/to/file.sh is a bash script that exists relative to the root of the Git repository. This is intended to simplify a yaml file by keeping complicated bash code in a file and allowing the readFile() command to read it as if it were written in the yaml file.

      Sample usage

      import net.gleske.jervis.tools.fileIO
//mystring path is relative to the root of the git repository
def mystring = 'readFile(/src/test/resources/sample_script.sh)'
def x = new fileIO()
println 'Is readFile? ' + x.isReadFile(mystring)
println 'Does the file exist?' + x.fileExists(mystring)
println 'What are the contents of the file?'
println x.readFile(mystring)

      • Method Detail

        • Boolean fileExists(String filestring)

          Checks the file path inside the parenthesis of 'readFile(/path/to/file.sh)' exists. Relative to the root of the Git repository, if the file /path/to/file.sh exists then it would return true. This method uses scmGit to determine the root of the Git repository.

          Parameters:
          filestring - A simple String that starts with readFile(, contains a path to a file, and ends with ).
          Returns:
          A boolean. If the file exists then it is true. If the the file doesn't exist or it is a malformed String then it returns false.

        • String readFile(String filestring)

          Read the contents of the file in a readFile() String. This method uses scmGit to determine the root of the Git repository.

          Parameters:
          filestring - Read the contents of /path/to/file.sh in the String readFile(/path/to/file.sh).
          Returns:
          A String which is the contents of the file read. If the filestring is malformed or the file doesn't exist then it simply returns the original string.
